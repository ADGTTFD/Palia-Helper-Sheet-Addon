🐟 Palia Fish Assistant

Google Apps Script application for searching the Palia Profit Guide.

The spreadsheet is the single source of truth. No fish names or recipes are hardcoded. Adding new rows or columns requires no code changes.

Features

  • Search fish by name
  • Search recipes by name
  • Automatically discovers new fish
  • Automatically discovers new recipes
  • Supports recipe variants
  • Highlights matching spreadsheet cells
  • Remembers previous lookup
  • Uses Google Sheets as the data source

Installation

  1. Open the Google Sheet.
  2. Select Extensions → Apps Script.
  3. Create these files:
Code.gs
Lookup.gs
Settings.gs

Sidebar.html
Styles.html
JavaScript.html

Readme.html

Copy the generated code into each file.

Save the project.

Run:

onOpen()

Authorise the project when prompted.

Reload the spreadsheet.

You should now see:

🐟 Fish Assistant

Using the Assistant

  1. Open 🐟 Fish Assistant → Open Assistant.
  2. Select a fish.
  3. Select a recipe.
  4. Click 🔍 Look Up.
  5. The matching worksheet cells are highlighted.

Spreadsheet Layout

The workbook must contain:

  • A header row containing recipe names.
  • One fish per row.
  • Recipe values beneath each recipe column.

Example:

Fish

Poke Bowl Dari
Poke Bowl SL / Sprout

Fish Stew Dari
Fish Stew SL / Sprout

Recipe Variants

Selecting Poke Bowl automatically searches: 

Poke Bowl Dari
Poke Bowl SL / Sprout

Selecting Fish Stew automatically searches: 

Fish Stew Dari
Fish Stew SL / Sprout

Additional variants are discovered automatically from the spreadsheet.

User Settings

Stored using PropertiesService.getUserProperties().

  • Theme
  • Highlight enabled
  • Remember last fish
  • Remember last recipe

Troubleshooting

Fish Assistant menu missing

Run:

onOpen()

Then reload the spreadsheet.

Sidebar won't load

Check the Apps Script execution log for errors.

Function not found

Ensure every generated file has been copied into the project.

No lookup results

  • Verify the fish exists.
  • Verify the recipe exists.
  • Ensure the worksheet name has not changed.
